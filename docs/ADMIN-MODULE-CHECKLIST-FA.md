# چک‌لیست ۱۸ مرحله‌ای هر ماژول ادمین

برای هر ماژول قبل از اعلام اتمام، همه موارد را تیک بزنید.

1. [ ] هدف ماژول و نقش‌های مجاز مشخص است
2. [ ] آیتم Sidebar (نام، آیکون، ترتیب، badge، permission)
3. [ ] Header (عنوان، Breadcrumb، دکمه‌ها، Quick Action)
4. [ ] صفحه لیست (ستون‌ها، sort، filter، pagination، search، selection، bulk)
5. [ ] Row actions (View/Edit/Delete/… حسب نیاز)
6. [ ] Bulk actions
7. [ ] Create form + validation + tooltip + خطا
8. [ ] Edit form (فیلدهای قفل/تاریخچه‌دار)
9. [ ] Print (در صورت نیاز)
10. [ ] Export (CSV/Excel/PDF حسب نیاز)
11. [ ] API: GET/POST/PUT/DELETE + zod + status codes
12. [ ] DB: فیلدها، relation، index، cascade
13. [ ] Security: permission روی API و UI
14. [ ] UI states: loading/empty/error/success
15. [ ] Responsive: desktop/tablet/mobile
16. [ ] Performance: pagination + lazy در صورت نیاز
17. [ ] Audit log برای تغییرات مهم
18. [ ] Scenarios: موفق، ویرایش، حذف، خطا
